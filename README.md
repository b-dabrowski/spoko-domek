# SPOKO DOMEK

Repozytorium startowe strony internetowej dla obiektu `SPOKO DOMEK` w Mikaszówce, przygotowane w `Next.js` pod hosting na `Vercel`.

## Status

Projekt jest gotowy do dalszej implementacji właściwej strony. W repo znajdują się:

- szkielet aplikacji `Next.js` z App Routerem,
- konfiguracja pod lokalny development i build produkcyjny,
- publiczne assety ze zdjęciami i logo obiektu,
- bazowe metadane i ekran startowy do dalszej rozbudowy.

## Uruchomienie

```bash
npm install
npm run dev
```

Aplikacja będzie dostępna pod adresem [http://localhost:3000](http://localhost:3000).

## Materiały

- zdjęcia źródłowe: `photos/`
- zdjęcia publiczne do użycia w aplikacji: `public/photos/`
- logo: `public/brand/`

## Następny krok

Zbudowanie finalnej strony sprzedażowej dla domku na podstawie dostarczonych informacji o obiekcie, wyposażeniu, okolicy i kontakcie.
